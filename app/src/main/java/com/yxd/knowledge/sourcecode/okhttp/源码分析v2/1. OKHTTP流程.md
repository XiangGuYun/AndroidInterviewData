# OkHttp流程

## 1. 大致流程图
![](img/b703828b.png)

关键：在Call执行同步/异步请求，以获取响应过程中，<font color=#dea32c>**分发器**</font>和<font color=#dea32c>**拦截器**</font>的工作流程。

> 分发器：完成请求调配（内部维护队列和线程池）。
> 
> 拦截器：完成请求流程（责任链模式）。
> > 分发器是管理者，拦截器是劳动者。
